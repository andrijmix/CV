Проєкти

SuperAgent — Інтелектуальний чат-агент на основі RAG

Огляд проєкту

SuperAgent — це інтелектуальний чат-агент, який використовує великі мовні моделі (LLM) разом із Retrieval-Augmented Generation (RAG) та Elasticsearch для надання точних і контекстно-залежних відповідей. Система маршрутизує запити користувачів до спеціалізованих агентів (FAQ-агент, агент уточнень, агент пошуку в Інтернеті), забезпечуючи якісні результати.

Цілі проєкту

  • Надавати надійні відповіді на поширені питання (FAQ).
  • Покращувати відповіді за допомогою RAG та семантичного пошуку в Elasticsearch.
  • Маршрутизувати повідомлення до відповідних спеціалізованих агентів.
  • Побудувати модульну та масштабовану архітектуру для AI-асистентів.

Архітектура

  • Backend: FastAPI для веб-маршрутизації та оркестрації сервісів.
  • Інтеграція LLM: LangChain / LangGraph для побудови та оркестрації агентів.
  • База знань: Elasticsearch (векторний пошук) для семантичного відбору.
  • Агенти: FAQ-агент, агент уточнень, агент пошуку та маршрутизатор повідомлень.
  • Основні компоненти: superAgent.py, qa_agent.py, rag_utils.py, elastic_utils/.

Технології

Python 3.10+, FastAPI, LangChain / LangGraph, Elasticsearch (Cloud), OpenAI API

Мій внесок

  • Розробив загальну архітектуру проєкту (architecture.txt).
  • Реалізував спеціалізованих агентів (FAQ-агент, маршрутизатор повідомлень) та RAG-утиліти.
  • Інтегрував Elasticsearch для семантичного пошуку та зберігання сесій чату.
  • Створив легкий веб-інтерфейс/роутери для обробки FAQ-запитів.
  • Написав документацію проєкту (README.md).

Результат

Повноцінний MVP чат-агента з підтримкою FAQ та RAG, здатний обробляти користувацькі запити у модульний і масштабований спосіб. Проєкт демонструє практичний досвід побудови систем оркестрації LLM, векторного пошуку та сучасних AI-рішень.
