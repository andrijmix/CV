SuperAgent — Intelligent RAG-based Chat Agent

Project Overview

SuperAgent is an intelligent chatbot that leverages Large Language Models (LLMs) with Retrieval-Augmented Generation (RAG) and Elasticsearch to provide accurate, context-aware answers. The system routes user queries to specialized agents (FAQ Agent, Clarification Agent, Web Search Agent) to ensure high-quality responses.

Project Goals

  • Provide reliable answers to frequently asked questions (FAQ).
  • Enhance responses with RAG using Elasticsearch for semantic retrieval.
  • Route messages to the appropriate specialized agents.
  • Build a modular and scalable architecture for AI assistants.

Architecture

  • Backend: FastAPI for web routing and service orchestration.
  • LLM Integration: LangChain / LangGraph for agent design and orchestration.
  • Knowledge Base: Elasticsearch (vector search) for semantic retrieval.
  • Agents: FAQ Agent, Clarification Agent, Web Search Agent, and a Message Router.
  • Core Components: superAgent.py, qa_agent.py, rag_utils.py, elastic_utils/.

Technologies

Python 3.10+, FastAPI, LangChain / LangGraph, Elasticsearch (Cloud), OpenAI API

My Contribution

  • Designed the overall project architecture (architecture.txt).
  • Developed specialized agents (FAQ Agent, Message Router) and RAG utilities.
  • Integrated Elasticsearch for semantic search and chat session storage.
  • Built a lightweight web interface/routers for FAQ queries.
  • Authored project documentation (README.md).

Result

A fully functional MVP chatbot with FAQ and RAG support capable of handling user queries in a modular and scalable way. The project demonstrates practical experience with LLM orchestration, vector search, and modern AI application design.
